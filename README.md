# bot ya.music from discord
Команды бота
1. /queue (префикс queue, например !queue <ссылка>)

Описание: Проигрывает все треки из альбома или плейлиста.
Аргументы:

<ссылка> — ссылка на альбом или плейлист Яндекс Музыки.
Примеры ссылок:

Альбом: https://music.yandex.ru/album/27856013

Плейлист: https://music.yandex.ru/users/USER/playlists/12345

Поведение:

Бот подключается к голосовому каналу автора команды.

Проигрывает каждый трек по очереди.

После окончания плейлиста бот автоматически отключается.

Можно прерывать воспроизведение (у тебя есть флаг break в коде, можно добавить команду для stop/pause).

2. /play (префикс play, например !play <ссылка>)

Описание: Проигрывает только один трек, независимо от того, часть ли это альбома или плейлиста.
Аргументы:

<ссылка> — ссылка на трек или альбом/плейлист.
Примеры ссылок:

Трек: https://music.yandex.ru/album/23265774/track/106618980

Альбом: https://music.yandex.ru/album/27856013 (тогда проиграется только первый трек)

Поведение:

Бот подключается к голосовому каналу автора команды.

Проигрывает только первый трек из ссылки.

После окончания трека бот автоматически отключается.

Ошибки и ответы

Если пользователь не в голосовом канале → сообщение из messages['discord']['no_voice_channel']

Если трек не найден → сообщение из messages['player']['no_track']

Любая непредвиденная ошибка → сообщение из messages['discord']['unknown']

Примечания для README

Префикс команд задается через .env переменную DISCORD_BOT_COMMAND_PREFIX.

Необходим токен Яндекс Музыки (YANDEX_MUSIC_TOKEN) и Discord бота (DISCORD_BOT_TOKEN).

Требуется наличие ffmpeg для воспроизведения треков через FFmpegPCMAudio.
