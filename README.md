# YaMusic Discord Bot

Бот для Discord с поддержкой проигрывания треков, альбомов и плейлистов с Яндекс Музыки.

---

## Префикс команд

Префикс задается в `.env` через переменную `DISCORD_BOT_COMMAND_PREFIX`.  
В примерах ниже используется `!` — замените на свой, если задан другой.

---

## Команды

### 1. `!queue <ссылка>`

**Описание:** Проигрывает все треки из альбома или плейлиста.  

**Примеры ссылок:**
- Альбом: `https://music.yandex.ru/album/27856013`
- Плейлист: `https://music.yandex.ru/users/USER/playlists/12345`

**Как работает:**
- Бот подключается к голосовому каналу автора команды.
- Проигрывает все треки по очереди.
- После окончания плейлиста бот автоматически отключается.

**Пример использования:**
```text
!queue https://music.yandex.ru/album/27856013
